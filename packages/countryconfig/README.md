# OpenCRVS Country Config Template

Basic design principles

- This template must be as simple and small as possible
  - All advanced examples are documented elsewhere, like in the OpenCRVS Farajaland repository
- Template only includes use cases we know most countries will need
- Countries never have to pull changes from this repository. When updating OpenCRVS toolkit, compiler should let the developers know about implementation details that might have changed or endpoints that must be implemented
